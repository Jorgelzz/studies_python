Esse repositório tem como principal função ser o meu campo de estudos, eu criei como inicio uma automação em python que cria um arquivo utilizando a data atual como nome e em seguida disponibiliza 3 opções base.
As opções inicias são 1º abrir no explorar, 2ª abrir com a IDE (sendo esse caso, o cursor) e por fim a 3º opção de sair.

É uma automação simples que utiliza Rich para mostrar o progresso e quando finalizado retorna o diretório onde foi criado e para adicionar um melhor visual no terminal.
